# MadLibs

AP CSA 1 MadLibs project by **Justin Zhou** and **Reza Bagheri**

## Installation

Use [Git](https://git-scm.com/downloads) to clone this repository.

```bash
git clone https://github.com/Codehc/MadLibs
```

## Running
Use Java to compile and run `MadLibs.java`
```bash
javac MadLibs.java
java MadLibs
```
